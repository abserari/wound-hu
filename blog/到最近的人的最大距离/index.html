到最近的人的最大距离

December 8, 2018 Abser

1 minute read

  • Twitter

  • Google+

  • Facebook

  • Reddit

  • LinkedIn

  • Email

问题描述

在一排座位（ seats）中，1 代表有人坐在座位上，0 代表座位上是空的。

至少有一个空座位，且至少有一人坐在座位上。

亚历克斯希望坐在一个能够使他与离他最近的人之间的距离达到最大化的座位上。

返回他到离他最近的人的最大距离。

示例 1

输入：[1,0,0,0,1,0,1]
输出：2
解释：
如果亚历克斯坐在第二个空位（seats[2]）上，他到离他最近的人的距离为 2 。
如果亚历克斯坐在其它任何一个空位上，他到离他最近的人的距离为 1 。
因此，他到离他最近的人的最大距离是 2

示例 2

输入：[1,0,0,0]
输出：3
解释： 
如果亚历克斯坐在最后一个座位上，他离最近的人有 3 个座位远。
这是可能的最大距离，所以答案是 3 。

思路

考虑三种情况，根据 0 个数来计算，最左边或最右边有最多的连续 0 ，那么结果就是相应 0 的个数，如果连续 0 在中间，考虑到连续 0 数量的奇偶，加上 1 再整除 2 。

实现

Go 版本

func maxDistToClosest(seats []int) int {
	maxDist, cnt := 0, 0
	for i := 0; i < len(seats) && seats[i] == 0; i++ {
		cnt++
	}
	maxDist = cnt
	for i := 0; i < len(seats); i++ {
		if seats[i] == 0 {
			cnt++
		} else {
			if maxDist < (cnt+1)/2 {
				maxDist = (cnt + 1) / 2
			}
			cnt = 0
		}
	}
	if maxDist < cnt {
		maxDist = cnt
	}
	return maxDist
}
